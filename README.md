# Daily Task Scheduler

A Java-based console application that helps users organize, manage, and track their daily tasks efficiently. The application allows users to add tasks with due times, view scheduled tasks, edit task details, mark tasks as completed, and monitor deadlines through a simple menu-driven interface.

## Features

- Add new tasks with due times
- View all scheduled tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Detect overdue tasks
- Receive warnings for upcoming tasks
- Dynamic task storage using ArrayList
- User-friendly console interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- LocalTime API
- Scanner Class
- VS Code

## How It Works

1. Launch the application.
2. Select an option from the menu.
3. Add tasks with a title and due time.
4. View all scheduled tasks.
5. Edit task details when required.
6. Mark tasks as completed.
7. Delete unwanted tasks.
8. Check for missed tasks and upcoming task warnings.
9. Exit the application.

## Key Features

### Task Creation
Users can create tasks by entering a task title and due time.

### Task Management
Tasks can be viewed, edited, completed, or deleted at any time.

### Missed Task Detection
The application automatically identifies tasks whose deadlines have passed.

### Upcoming Task Alerts
Users receive warnings when tasks are due within the next ten minutes.

### Dynamic Storage
Tasks are stored using ArrayList, allowing efficient insertion, deletion, and updates.

### Console-Based Interface
A simple menu-driven interface ensures easy navigation and operation.
